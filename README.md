# Homebrew Tap for KAI OS

Install KAI OS with Homebrew:

```bash
brew tap morning-verlu/tap
brew install kaios
kaios doctor
kaios run "analyze crypto market"
kaios ps latest
```

KAI OS is an AI Agent Operating System in Kotlin.

- Website: https://morning-verlu.github.io/KAI/
- Repository: https://github.com/morning-verlu/KAI
